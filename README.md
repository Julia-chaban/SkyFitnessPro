
SkyFitnessPro — Фитнес-приложение для онлайн-тренировок
SkyFitnessPro — это современное веб-приложение для онлайн-тренировок, разработанное на React с использованием TypeScript. Приложение позволяет пользователям выбирать фитнес-курсы, отслеживать свой прогресс и проходить тренировки с видеоуроками.

Технологии
Проект построен на современном стеке технологий: фронтенд на React и TypeScript с использованием React Router для навигации и CSS Modules для стилизации. Бэкенд реализован на Node.js с Express, в качестве базы данных используется SQLite через библиотеку better-sqlite3. API построено по принципам REST с JWT-авторизацией. Сборка проекта осуществляется с помощью Create React App.

Функциональность
Приложение предоставляет полный цикл работы с фитнес-тренировками. Пользователь может зарегистрироваться, указав email и пароль с требованиями безопасности (не менее 6 символов, минимум два спецсимвола и одна заглавная буква), после чего войти в систему. На главной странице отображаются все доступные курсы, каждый из которых содержит подробное описание, продолжительность и уровень сложности. Авторизованный пользователь может добавить понравившийся курс в свой профиль и приступить к тренировкам.

Каждая тренировка включает видеоурок, встроенный с YouTube, и набор упражнений с указанием количества повторений. Пользователь может отмечать свой прогресс, вводя количество выполненных повторений, система автоматически рассчитывает процент выполнения и отображает его на шкале прогресса. Все данные о прогрессе сохраняются и доступны при последующих заходах.

В личном профиле пользователь видит свои данные, список приобретенных курсов с прогрессом по каждому, а также может удалить курс из своего списка. Выход из аккаунта осуществляется через модальное окно профиля.

Адаптивный дизайн
Приложение полностью адаптировано для всех типов устройств. Используются медиа-запросы для ширины экрана 375px (мобильные телефоны), 768px (планшеты), 1024px, 1280px и 1440px (различные размеры десктопов). Все элементы интерфейса корректно масштабируются и перестраиваются в зависимости от размера экрана.

Установка и запуск
Для запуска проекта необходимо склонировать репозиторий, установить зависимости командой npm install, затем запустить API сервер командой npm run api и в отдельном терминале запустить React приложение командой npm start. Приложение будет доступно по адресу http://localhost:3000, API сервер — на порту 3002.

Структура API
API включает следующие эндпоинты: регистрация и вход пользователя, получение данных текущего пользователя, список всех курсов и детали конкретного курса, добавление и удаление курсов из профиля, получение списка тренировок курса и деталей конкретной тренировки, сохранение прогресса и получение данных о прогрессе пользователя. Все защищенные маршруты требуют передачи JWT-токена в заголовке Authorization.
